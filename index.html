DatadumpTool

A Python-based command-line utility for deploying, querying, and generating statistics from MongoDB datadumps.

Download this project as a .zip file Download this project as a tar.gz file

What is the Datadump Tool?

This tool was developed to automate and accelerate the processing of data dumps from a MongoDB database. It focuses on statistical analysis of the graph structure described in the data dumps.

How do I use the tool?

First, be sure to install the following third-party libraries, preferably in a Python virtual environment.

Required Third-Party Libraries

  • pymongo
  • pygraph
  • networkx
  • numpy
  • scipy
  • matplotlib

Once all the required libraries are installed, execute the main script as follows:

$ python datadump_tool.py

As the script is running, you will see the following prompt:

(datadump-tool)

You may type 'help' at the prompt to see a list of the available commands:

(datadump-tool) help

Documented commands (type help <topic>):
========================================
generate_dump  load_dump  deploy_dump  clear_mongo  stats  print_root_node_data  print_authors 
find_leaf_nodes  find_root_nodes  calc_graph_scc  show_types  exit
